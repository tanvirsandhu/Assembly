Lab 4: Sorting Integers

----------
DESCRIPTION

In this lab, the program takes up to eight user-given program arguments in hexadecimal. It then prints the arguments, the decimal conversion, and the sorted list of decimals, all with appropriate labelling.

----------
FILES 

Lab4.asm

This file is the complete work of the lab, where the user can do the above commands. It can be opened using MARS.
-
Diagram.pdf

This file holds the block diagram which has a description for the process of reading program arguments, converting the ASCII strings to decimal values, sorting and printing. 
----------
INSTRUCTIONS

This program is intended to run on the application MARS and can be downloaded on Windows or Macintosh.
