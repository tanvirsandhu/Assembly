Lab 3: ASCII-risks

----------
DESCRIPTION

In this lab, the user can give a number greater than 0 and the output will be a pyramid of the numbers in ascending order, spanning the specified number of lines. The space between the numbers will be filled with asterisks. If the user puts an invalid number, such as 0 or any negative number, the system will output "Invalid Entry!"

----------
FILES 

Lab3.asm

This file is the complete work of the lab, where the user can do the above commands. It can be opened using MARS. 
----------
INSTRUCTIONS

This program is intended to run on the application MARS and can be downloaded on Windows or Macintosh.
