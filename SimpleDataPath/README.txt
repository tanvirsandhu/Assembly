Lab 1: Intro to Logic Simulation

----------
DESCRIPTION

In this lab, the user can send keypad values to registers 0-3. In addition, they can create the bitwise right rotation of the values that they've give to registers, by selecting the ALU output, and assigning it to a register by pressing update and putting the destination on the write address switches. They can clear all the registers simply by pressing clear. 

----------
FILES 

Lab2.lgi

This file is the complete work of the lab, where the user can do the above commands. It can be opened using Multimedia Logic.

----------
INSTRUCTIONS

This program is intended to run on the Windows application Multimedia Logic, but can be opened on Macintosh using WineBottler.
